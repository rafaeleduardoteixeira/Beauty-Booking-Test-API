# Beauty Booking API

A REST API built with JSON Server for the Arionkoder test project.

## Prerequisites

- Node.js >= 18.0.0
- npm (comes with Node.js)

## Installation

1. Install dependencies:

```bash
npm install
```

3. Create a `db.json` file in the root directory with your data structure (if not already present).

## Running the Server

### Development Mode (with auto-reload)

```bash
npm run dev
```
